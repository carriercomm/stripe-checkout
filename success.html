Payment received

Upon reaching this page, you'll want to verify that the charge token is indeed a valid paid charge created in the last few seconds. In your server-side pseudocode:


s = Stripe.client('PRIVATE_API_KEY')
charge = s.retrieve_charge(id: $GET['charge'])
if (charge.paid && (now() - created() < 60)) {
  // do good things
} else {
  // it's a trap!
}
